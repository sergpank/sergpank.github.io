# Where are CodeDeploy logs?
Default location for CodeDeploy logs on `Amazon Linux` is:  
``` bash
/opt/codedeploy-agent/deployment-root/...
# or maybe :
/var/log/aws/codedeploy-agent/...
# depends on installed version
```
